# Gym Trainer Management System

A Windows Forms application built with C# and .NET Framework 4.7.2 designed to manage a gym facility.

## Features

- **Role-based Access Control**: Separate interfaces for Admins, Owners, Trainers, and Members.
- **User Management**: Registration modules for Owners, Trainers, and Members.
- **Fitness Management**: Functionality to manage Workout Plans and Diet Plans.
- **Custom UI**: Uses custom components (e.g., `Button_WOC`) for an enhanced user interface.

## Technologies Used

- C#
- .NET Framework 4.7.2
- Windows Forms (WinForms)

## Setup

Open the solution or project file (`DB.csproj`) in Visual Studio and build the project to restore dependencies and run the application.
